# my-portfolio
My personal portfolio website was built with HTML, CSS and JavaScript
This is my personal portfolio website, designed and developed to showcase my skills, projects, experience, and journey as a software developer. The website serves as a central hub where visitors, recruiters, and collaborators can learn more about me, explore my work, and easily get in touch.

The platform highlights my technical abilities, including frontend and backend development, UI/UX design principles, and project-building experience. Each section of the portfolio is carefully structured to present information in a clean, modern, and user-friendly way, ensuring smooth navigation and an engaging user experience across all devices.

The website features a fully responsive design optimised for desktop, tablet, and mobile screens. It includes sections such as an about me page, skills overview, project showcase with live demos and source code links, and a contact section for direct communication.

This portfolio reflects my passion for building meaningful and visually appealing digital experiences, as well as my ongoing growth in software development. It is regularly updated as I learn new technologies and complete more projects.
